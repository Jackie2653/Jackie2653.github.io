使用Hexo＋Github搭建个人博客

发表于 2020-02-20 更新于 2020-03-04 分类于 Tech 阅读次数： Valine：
本文字数： 1.6k

只需简单三步，小白也能搭建属于自己的个人博客。

因博主最初开始搭建时也是一脸懵逼，查阅了很多资料，官方文档太繁琐，网上资料又参差不齐，且有的版本过低，故写此文记录踩坑，提供参考。

工具为hexo＋github，环境为windows10。如有错误，欢迎留言或联系作者，也欢迎大佬指正

环境准备

Git安装

进入Git官网.

根据自己PC信息选择下载，如博主下载的是win 64

提示：Git安装若没有梯子可能相对较慢，需要耐心等待

也可以直接通过网盘下载

关于git

git是一个开源的分布式版本控制系统，可以有效、高速地处理从很小到非常大的项目版本管理。 Git 是 [Linus Torvalds](https://baike.baidu.com/item/Linus Torvalds/9336769) 为了帮助管理 Linux 内核开发而开发的一个开放源码的版本控制软件。

简单易懂的git教程 by 廖雪峰

Node.js安装

进入Node.js官网.
下载左侧长期支持版并安装，无脑下一步即可

npm是一个包管理工具，nodejs已集成，但由于npm在国内比较慢，所以推荐安装cnpm淘宝镜像

(博主最初就是因为用npm安装相关插件等的怀疑人生……..)

Checkout

查看版本信息以检查是否安装成功

若出现版本信息，则安装成功，若无版本信息，很有可能是你在安装时敲错命令，试着重新敲一遍，确保安装成功
